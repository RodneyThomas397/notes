<!DOCTYPE html>:

This is the document type declaration. It ensures that every web browser parses(1) the webpage the same.

<head></head>:

This is where you would put the meta tags and other important information for the webpage. Usually information that is in the background.

<meta>:

This defines the metadata about the webpage, which is the information about the webpage, such as the description. Metadata is not put in front of the user on the page.

  1. <meta name=”author” content=”John Doe”>: Describes the author of the page.
  2. <meta name=”description” content=”Homepage of Website”>: Describes the page.
  3. <meta name=”” content=’”>: Describes the keywords for a webpage, which can be used in search engines to help people search for your website.
  4. <meta http-equiv=”refresh” content=”10”>: Refreshes the document based on the number in the “content”.
  5. <meta name="viewport" content="width=device-width, initial-scale=1.0">: Makes web pages look decent / normal on all devices and screen resolutions.

<!-- Put comments here -->:

This is how you make comments in HTML, these will be ignored in the parsing of the webpage, they are there for the developer.

<title></title>:

This is where the title of the webpage goes, usually visible in the tab in the web browser on the top, or in the history.

<body></body>:

This is where all the main content goes that will be visible to the users.

<script></script>:

This is a javascript client built into your web page, and will be automatically loaded if it is inside of there.

  1. <script type=”text/javascript” src=”game.js”></script>: This is referring to the javascript file that is in the same directory as your index.html file. It will execute everything inside of that file.
  2. <script type=”text/javascript”>console.log(“Hello World”);</script>: This will log Hello World into the console, accessible by holding
